Publicando um site no GitHub Pages

Publicado em 22/04/2026 · #Git #Deploy

O GitHub Pages transforma um repositório em um site publicado, de graça, direto do seu próprio GitHub.

1. Crie o repositório

Pode ser um repositório novo ou um que você já tenha, com os arquivos .html, .css e .js do seu site.

2. Ative o GitHub Pages

  1. Vá em Settings do repositório
  2. No menu lateral, clique em Pages
  3. Em "Source", selecione a branch main e a pasta /root
  4. Clique em Save

3. Acesse o site publicado

Depois de alguns minutos, o site fica disponível em um endereço no formato https://seu-usuario.github.io/nome-do-repositorio/.

Dica

Toda vez que você fizer um novo commit na branch configurada, o GitHub Pages atualiza o site publicado automaticamente — não precisa reativar nada.
