Como montar um bento grid em CSS

Publicado em 01/07/2026 · #CSS #FrontEnd

Um "bento grid" é uma grade de blocos de tamanhos variados, como uma marmita japonesa — daí o nome. Em CSS puro, dá pra montar isso com poucas linhas usando display: grid.

1. A base do grid

Comece definindo quantas colunas o grid deve ter e o espaço entre os itens:

.bento-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    gap: 25px;
}

O auto-fill com minmax() faz o grid quebrar linha sozinho conforme a largura da tela, sem precisar de media queries manuais.

2. Cards de tamanhos diferentes

Para dar variedade ao bento, alguns itens podem ocupar mais de uma coluna ou linha:

.destaque {
    grid-column: span 2;
    grid-row: span 2;
}

3. Responsividade

Como o grid já se adapta sozinho com auto-fill, normalmente só é preciso ajustar o minmax() em telas muito pequenas.

Conclusão

Com poucas regras de CSS Grid já dá pra montar um layout parecido com o deste site: cards uniformes, responsivos, sem JavaScript nenhum.
